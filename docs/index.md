# Dental Telegram Bot MVP

Гибридный Telegram-бот для стоматологической клиники: пациент может начать с кнопок или свободной фразы, а критичные действия (запись, перенос, отмена) выполняются только через кнопки.

## Возможности

| Область | Описание |
|---------|----------|
| **Запись на приём** | Выбор услуги и слота кнопками |
| **Текстовый intake** | AI маршрутизирует жалобу и показывает реальные слоты |
| **Мои записи** | Просмотр, перенос и отмена |
| **Цены и контакты** | Данные из профиля клиники |
| **Быстрая регистрация** | Один клик при первом входе |
| **Админ-панель** | Дашборд, записи, демо-сценарий |
| **Напоминания** | Worker отправляет уведомления за 24 часа |
| **Безопасность AI** | Отказ от диагнозов, проверка галлюцинаций |

## Принципы продукта

1. **Гибридный UI** — свободный текст на входе, кнопка на подтверждении.
2. **AI только для маршрутизации** — не создаёт записи и не ставит диагнозы.
3. **Источник правды** — услуги, цены, врачи и слоты берутся из Core API.
4. **Событийная архитектура** — ключевые действия публикуются в NATS JetStream.

## Стек

- **Python 3.12**, FastAPI, aiogram, SQLAlchemy, PostgreSQL
- **Redis** — rate limiting
- **NATS JetStream** — события и фоновые задачи
- **Docker Compose** — локальный и демо-запуск
- **MkDocs + Material** — документация

## Быстрый старт

```bash
cp .env.example .env
# Заполните TELEGRAM_BOT_TOKEN, ADMIN_TOKEN
docker compose up --build
```
